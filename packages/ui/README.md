# packages/ui

Biblioteca de componentes visuais compartilhados do Icarus Atend.

## Objetivo nesta fase

Concentrar o design system inicial e reduzir duplicação de UI entre apps.

## Tokens

- Cores semânticas (primária, secundária, fundo, superfície, destaque, estados)
- Espaçamentos
- Tipografia
- Bordas e sombras

## Componentes base

- Button
- Input
- Textarea
- Card
- Badge
- Tabs
- Modal
- Avatar
- EmptyState
- SectionHeader
- StatCard
